# eCFR Analyzer

## Requirements
* Go 1.21+

## Run locally
`go run server.go`

## Environment Variables

```
export ECFR_TOKEN_SECRET
export ECFR_DB_USER
export ECFR_DB_PASS
export ECFR_DB_HOST
export ECFR_DB_PORT
export ECFR_DB_NAME
export ECFR_DB_INSTANCE_CONNECTION_NAME
export ECFR_DEVELOPMENT
```
